Rodea The Sky Soldier é um game que mistura mecânicas de shooter e plataforma. Desenvolvido por Yuji Naka, o criador do Sonic The Hedgehog, Rodea tinha tudo para ser um grande título para Nintendo Wii U e 3DS. Mas o desenvolvimento conturbado aliado às péssimas conversões, fazem este game ser deixado para trás. Entretanto, a versão bônus do Wii salva o pacote. Entretanto, graças a uma produção repleta de contratempos, o game, que estava previsto primeiramente para Nintendo Wii foi portado para 3DS e Wii U de forma errada por uma empresa terceirizada.O game estava previsto para 2010, mas com o 3DS e Wii U chegando ao mercado, a produtora Prope, desistiu do lançamento no Wii e levou o game para as novas plataformas da Nintendo. O problema é que essa tarefa não ficou a cargo da Prope, mas da Kadokawa.A decisão foi tomada de uma hora para outra, mas a conversão dos controles levou bastante tempo e o resultado final foi decepcionante. Dessa forma, a publicadora incluiu a versão original do Nintendo Wii para quem comprar cópias físicas de Rodea The Sky Soldier para 3DS e Wii U.Controles ruinsRodea The Sky Soldier, como o próprio nome sugere, é um jogo sobre voar e atacar inimigos em batalhas aéreas. Em sua versão para Wii, a mecânica de atacar os adversários e saltar de um cenário para o outro é bastante rápida, graças a uma jogabilidade que mistura shooter com plataforma.Porém, no Wii U, os controles são bem diferentes. Ao saltar, o jogador deve mirar usando o joystick e isso quebra o frenesi dos combates. Na realidade, o voo é tão lento que é melhor enfrentar os inimigos em solo e depois voar para explorar o ambiente.Controles ruins estragam a diversão da versão do jogo para Wii U (Foto: Divulgação / NIS America)Muitos dos problemas relacionados ao voo está no controle da câmera. Ela só está habilitada para livre movimentação durante o voo e exige muito treino: saltar, habilitar a câmera livre com o analógico direito, atacar os inimigos e controlar o voo com o analógico esquerdo. É muita coisa para um curto espaço de tempo, já que há um limitador de tempo para o voo.O resultado é uma confusão de comandos em que o usuário simplesmente prefere apenas voar reto, como se estivesse saltando de uma plataforma para outra. Só depois de um longo tempo melhorando o personagem é que o voo irá durar mais e é possível utilizar a movimentação da câmera sem ser atrapalhado pela queda.Visual de jogo portátilRodea The Sky Soldier para Nintendo Wii U é um port da versão para Nintendo 3DS. O game possui texturas com cores mortas e quantidade de polígonos que simplesmente não condiz com um game para console de mesa. Mas Rodea apresenta vários problemas de performance demonstrando que foi muito mal portado.Se possível, pegue a versão para WiiExiste inúmeras diferenças entre Rodea The Sky Soldier para Nintendo Wii U e Nintendo Wii. A versão para Wii é mais colorida e possui gameplay mais rápido. Um grande diferencial desta versão é que ela não possui limite de voo como para Wii U.A versão do Wii é tudo aquilo que o jogo realmente deveria ser, uma mistura entre Sonic e Nights com uma jogabilidade rápida e viciante. No Japão, os jogadores perceberam isso imediatamente e passaram a ignorar a versão do Wii U. Tanto é, que por lá, a versão vendida de forma oficial é a versão do Wii, enquanto a do Wii U vai como bônus na compra.ConclusãoRodea The Sky Soldier é uma demonstração de como leva tempo para uma produtora Third Party compreender os controles únicos criados pela Nintendo. A versão para 3Ds e Wii U decepciona muito e a jogabilidade irrita em vários momentos. Já a versão para Wii é um game completamente diferente.
